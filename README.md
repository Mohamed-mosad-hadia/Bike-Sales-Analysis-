Bike Buyers Dataset 
This dataset has details of 1000 users from different backgrounds and whether or not they buy a bike
Columns -

 - ID
- Marital Status
  
 - Gender
   
  - Income
   
  - Children
  
  - Education
  
  - Occupation
  
  - Home Owner
  
  - Cars
  
  - Commute Distance
  
  - Region
  
  - Age
  
  - Purchased Bike
